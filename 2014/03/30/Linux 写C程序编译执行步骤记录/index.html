Linux 写C程序编译执行步骤记录

  1. 选定好一个目录（或者mkdir 文件名）
  2. vim cFile.c
  3. vim 开始编辑代码
    int main(){
    return 0;
    }
    //x 保存并退出;
  4. 在当前目录执行 gcc cFile.c (gcc 文件名 如果没有gcc自行安装) 会生成一个 .out 输出文件
  5. 我可也可以使用 gcc cFile.c -o 文件名 生成一个我们想要的输出文件 注意这里生成的文件只有文件名
  6. 执行编译后的文件需要 ./ .out( 你的文件名) 或者是你用 gcc -o选项 输出的文件名 ./文件名
  7. 因为刚才的那段代码 return 0 所以看不到 你可以输入 echo $? 就看到 0 你也可以试试改变 return 100 然后重新编译执行.
